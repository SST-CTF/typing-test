## Description
The best way to test typing speed and accuracy for a user provided string. This JS + HTML combo can be run server or client side.
### Features
- Test any string
- Can run local or server
- Copy / Paste Protection
- Gross and Net WPM
- Error checker
- Timer
- Looks pretty!
